#LagClearTimer
Description: A client-side mod that creates a timer to be used to track the Lag Clear on a Minecraft Server.

Current Features:
1. Counts down from 7 minutes and 40 seconds
2. Resets when the Lag Clear message is received in chat
3. Can be hidden / shown using a keybind (Default: Grave Key)
4. Command to change the color of the GUI
5. Full support of Default Colors
6. Full support of Hexadecimal Colors

Future Features:
1. Command to change the location of the timer
2. Command to change the target message
3. Command to change the timer countdown

Possible Features:
1. GUI for the color of the timer text
2. GUI for the location of the timer
3. GUI to handle target message and countdown
